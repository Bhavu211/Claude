# Career Copilot — Voiceover Script
For `career-copilot-demo.mp4` (2:22 / 142s, silent screen recording)

Timestamps are approximate guide markers, not hard sync points — the live
pipeline run (0:32–0:55) can shift a few seconds run to run. Read at a
measured, confident pace; it's fine to leave silence over the sections
where the visual is doing the work (the live graph filling in, tables
scrolling). Total spoken content is ~127s inside a 142s video, so there's
built-in breathing room.

### Recording notes
- Total runtime target: ~140s of narration + natural pauses fits the 142s video comfortably.
- The live-execution section (0:32–0:55) is the one place timing can drift — if your take runs short, just pause and let the graph finish before starting the "In under a minute, it's done" line.
- Keep tone plain and confident, not salesy — the visuals (real graph, real data) are already doing the selling.
